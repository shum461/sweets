# Make a pooled disposition table

Make a pooled disposition table

## Usage

``` r
pool_disposition(disp_table)
```

## Arguments

- disp_table:

  an object of class 'disposition_table'

## Value

A pooled disposition table.

## Examples

``` r
if (FALSE) { # \dontrun{
pool_disposition(disp_table = my_disposition_table)
} # }
```
