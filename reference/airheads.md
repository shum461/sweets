# Track QC notes and findings

Retrieve findings and notes from QC program and generate QC email
FINDING: this is a finding or NOTE: this is a note

## Usage

``` r
airheads(qc_data_path = NULL, qc_program_path = NULL)
```

## Arguments

- qc_data_path:

  path to qc dataset. Auto loads qc_data_path if already in environment.

- qc_program_path:

  path to qc code file with comments. Default opens active tab

## Examples

``` r
if (FALSE) { # \dontrun{
airheads()
} # }

```
